**Environment**
- macOS Big Sur 11.5.2
- Xcode 13.1
- SwiftUI 2
- Deployment Target 15.0

**Feature**
- Multi-platform app for iOS/iPadOS/macOS
- Using Matched Geometry Effect to create transition
